Q. Which code we can use to scale an image width?

A: Scale_Width()

B: scaleToWidth()

C: scale(width)

D: scaleWidth(X)

Answer:Option 2

Explaination:Option 2 is the answer. scaleToWidth() is the predefine function which can be used to set new width value on the image. Eg. block_image_object.scaleToWidth(block_image_width);

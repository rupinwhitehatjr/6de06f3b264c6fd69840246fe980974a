Q. Identify the correct output, if we set the following behaviour

A: A. Can change the colour of only one sprite to a random colour.

B: B. Can change the colour of all the specified sprites to a random colour.

C: C. Can change the colour of all the specified sprites to red

D: D. Can change the colour of only one sprite to red

Answer:Option 3 (text)

Explaination:As we specify "this sprite" in the behaviour, it can change the colour of all the sprites for which this behaviour is set

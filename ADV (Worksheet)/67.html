Q. My car is not moving properly. Can you please help me to move it correctly?

A: There's no error, the car will move properly.

B: You need to change the "while" to "when" in all the events.

C: There is no event on the press of the right arrow key. Change the first left arrow event to right arrow event.

D: There is no event on the press of the right arrow key. Change the second left arrow event to right arrow event.

Answer:Option 3 (text)

Explaination:Option 3 is the answer. The car is not moving properly because nothing is happening when right arrow key is pressed. And also for [while left pressed] there are two different event blocks attached that will cause an ambiguity as the car will try to move in both the east & west directions while the left arrow key is pressed. So to move the car in all 4 direction we need to use 4 different keys. When we replace the first left arrow key event with right arrow key, the car moves in the east direction i.e. in the right direction.

Q. I want the output to be drawn on centre, but with my current code I get the output at the left top corner. Help me to correct the code to get the desired output.

A: In Line 8 , turnRight angle should be 90 instead of 1.

B: In Line 3 moveTo() function should be removed.

C: In Line 3, moveTo(30,100) should be changed to moveTo(155,250)

D: In Line 6, Penwidth value should be 5.

Answer:Option 3 (text)

Explaination:As mentioned in the question the pattern is drawn on the top left corner because we have set the position of the pen as (30, 100). So to draw a pattern somewhere in the middle of the screen we need to change the pen position using moveTo(155, 250)

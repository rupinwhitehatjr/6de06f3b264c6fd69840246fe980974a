Q. What is the output for this code?

A: The green sprite will stop if the yellow sprite touches it.

B: The yellow sprite will move up when clicked and the yellow sprite will start jittering when green sprite touches it.

C: The yellow sprite will move up when clicked and the green sprite will start jittering when yellow sprite touches it.

D: The green sprite will move south when clicked.

Answer:Option 2 (text)

Explaination:There's no block for stop hence option 1 is incorrect, option 4 is incorrect because there's no click event on green sprite. Option 3 is incorrect because yellow sprite should jitter when green sprite touches it and not the green sprite should jitter

Q. Bhav is trying to create a Maths quiz game, The program generates 2 random numbers between 2(n1) and 8(n2) and the actor in the game asks " what is n1 * n2 ? " instead of the actual numbers. Identify the mistake in the code because of which the program is not working as expected.

A: 1) We should give the random integer block to select numbers between “n1” and “n2” instead of 2 and 8.

B: 2) The actor 1 say block should be “what is 2(n1 value ) multiplied by 8(n2 value)”?

C: 3) The “n1” and “n2” text blocks in actor 1 say block should be replaced with n1 and n2 variables.

D: 4) We should declare the variables as “number1” and “number2” instead of n1 and n2.

Option 3 (text)

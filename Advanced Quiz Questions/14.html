Q. How many times we need to repeat the binary code "10110" to complete the below picture?

A: 6.0

B: 10.0

C: 13.0

D: 12.0

Answer:Option 3 (text)

Explaination:There is a 8*8 grid which contains 64 cells and the identified string is of length 5, hence it should be repeated 64/5= 12.8 i.e. 13 times.

Q. Numbers are assigned to steps that have to followed to create an intent, but are jumbled. Choose the right sequence of steps to be done, to create an intent.

A: 1, 2, 3, 4, 5

B: 1, 4, 5, 2, 3

C: 3, 4, 5, 1, 2

D: 1, 3, 2, 4, 5

Answer:Option 4 (text)

Explaination:To create an Intent we need to follow following sequence- Name the intent->Add training phrase-> Add response for the intent -> click on save -> Test the agent for newly created intent.

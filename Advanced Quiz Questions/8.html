Q. Debug the code with the help of options given below to draw the shapes shown on the left.

A: No changes needed, the code block given is correct

B: Add "Jumpbackward by 100 pixels" before the second repeat loop and then "Jumpforward by 200 pixels" block before the third Repeat loop.

C: Add "MoveForward by 50 pixels" block between Repeat loops

D: None of the above

Answer:Option 2 (text)

Explaination:if there is no jump block after the repeat loops then the bases of all the triangles will be connected, hence we need to add a jump block after every repeat block so that there is some space between the triangles.

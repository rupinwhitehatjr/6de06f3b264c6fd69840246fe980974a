Q. Which coordinates need to be changed to make the angry bird reach the silly pig?

A: X coordinates need to change and Y remains same

B: Both X and Y needs to be changed

C: Y coordinates need to change and X remains same

D: None of the above.

Answer:Option 1 (text)

Explaination:Angry bird needs to move horizontally so Y coordinate will remain the same and the X coordinate will change.

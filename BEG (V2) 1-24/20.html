Q. What will we do to transform Output-1 to Output-2 ?

A: Use Repeat Block

B: Use Set Color Block

C: Use Set Pattern Block

D: None of the above

Answer:Option 2 (Text)

Explaination:The set Color block is used to set or change the color of the artist's brush and hence is the correct answer. In this case the colour will change from black to red.

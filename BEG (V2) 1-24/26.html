Q. When we want to move the mouse horizontally (sleeping line) on the display area then which value will change?

A: "X" Value

B: "Y" Value

C: Both "X" and "Y" Value changes

D: None of the above

Answer:Option 1 (text)

Explaination:If we move our mouse along a horizontal line, the X value changes while the Y value stays the same.

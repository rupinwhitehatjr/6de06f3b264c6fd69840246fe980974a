Q. Which of the following options is not true?

Which of the following options is not true?image

A: Algorithm (logic flow) is created by breaking the problem into simple steps.

B: Order of instructions in an algorithm is important.

C: Instructions must be precise or exact.

D: Computer follows instructions in any language.

Answer:Option 4 (text)

Explaination:Computer only follows binary language i.e. 1s and 0s. Hence option 4 is the correct option.

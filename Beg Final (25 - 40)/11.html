Binary is a way of representing information using only two numbers. Today we learned how to use 2 numbers to make images. One was for off and the other was for on. Choose the correct optionimage

A: The number 0 as code for "on" and the number 1 as code for "off".

B: The number 1 as code for "off" and the number 2 as code for "on".

C: The number 0 as code for "off" and the number 1 as code for "on".

D: None of the above.

Answer:Option C (Text)

Explaination:Binary is a code system that represents number 0 as OFF and 1 as ON.

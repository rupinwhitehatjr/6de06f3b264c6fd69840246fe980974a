Q. Look at the code given, what will happen when you click on carrot sprite?

A: The carrot will start growing when clicked and stop when clicked again.

B: The carrot will never start growing.

C: The carrot will start spinning when it is clicked.

D: Both b and c

Answer:Option 1 (text)

Explaination:when Sprite Carrot is clicked, Carrot starts growing, and the boolean value of i is then set to false. When the carrot is clicked again, the Carrot stops growing, and boolean value of i is set to true.

Q. Which of the below statements is false for pixels?

Which of the below statements is false for pixels?image

A: Pixels show the length of a line we want to draw on the screen.

B: To draw shorter lines, we reduce the number of pixels and to draw longer lines, we increase the number of pixels in our code blocks

C: Pixels are units to measure the size of an image

D: Pixels determine the color of the line drawn

Answer:Option D (Text)

Explaination:A pixel is a single point or dot in a graphic image, and is used as a unit of measurement for drawing and images. It does not determine color in coding blocks.

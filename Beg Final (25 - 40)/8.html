Q. Choose to correct syntax that should be used to explain “ Bucket is not full, keep adding water”

Choose to correct syntax that should be used to explain “ Bucket is not full, keep adding water”image

A: while (Bucket does not have space) { Keep adding water }

B: while (Bucket has space) { Do not add water }

C: while (Bucket has space) { Keep adding water }

D: while (Bucket have water) { Keep adding water }

Answer:Option C (Text)

Explaination:According to option 3, it will keep adding water until the bucket has space, and then stop.

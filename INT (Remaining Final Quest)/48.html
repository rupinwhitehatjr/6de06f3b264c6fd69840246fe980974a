Q. What happen while UP arrow key is pressed, and kept pressed?

A: Plane sprite will move up by 20 pixels.

B: Plane sprite will move north and also Edges of play area will block sprite.

C: Plane sprite will go south.

D: Nothing, code will give error.

Answer:Option 2 (text)

Explaination:If “While key pressed” block is used, the event will be in a continuous state until we unpress the key.

Q. What will be the output of below code snippet?

A: Box collided gets printed in console if box1 touches box2 by moving mouse.

B: No Box Collided gets printed in console if box1 touches box2 by moving mouse

C: An Error will occur

D: Both sprites box1 and box2 will remain in the same position, nothing will get printed.

Answer:Option 1 (text)

Explaination:Using isTouching(), we can check if box1 touches box2 according to mouse movement. So while moving box1 according to mouse the isTouching() block will be executed and "Box collided" statement will be printed if it touches box2 and if boxes do not touch each other "No Box collided" will get printed in console.

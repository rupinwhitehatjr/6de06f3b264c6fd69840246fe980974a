Q. Which of the following statement is true, if we run the following code snippet

A: If user has pressed left arrow key, the ball will move to right.

B: If user has pressed left arrow key, the ball will move to left

C: If user has pressed left arrow key, the ball will not move at all.

D: A syntax error will be thrown

Answer:Option 1 (text)

Explaination:We can observe in code that we have set the velocity in X as positive and velocity in Y as zero, which means that the object will move towards the right

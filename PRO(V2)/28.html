Q. While designing the pong game, we made an error in which the ball bounces off all the walls, select the correct option to fix this issue

A: Remove ball.bounceOff(topEdge); ball.bounceOff(bottomEdge);

B: Remove ball.bounceOff(leftEdge); ball.bounce(rightEdge);

C: Remove ball.bounceOff(playerPaddle); ball.bounceOff(computerPaddle);

D: None of the above

Answer:Option 2 (text)

Explaination:In the code snippet we can observe that the ball bounces off all the edges, but for the pong game we do not want the ball to bounce off left and right edges. So we have to remove ball.bounceOff(leftEdge); and ball.bounceOff(rightEdge);

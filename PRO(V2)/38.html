While giving velocity to our ball, we want to move the ball untill the key is pressed in left or right direction. Choose the correct code to make the ball move untill the key is kept pressed.image

A: if(keyWentDown("right")) { ball.x = ball.x + 2; } if(keyWentDown("left")) { ball.x = ball.x - 2; }

B: if(keyDown("right")) { ball.x = ball.x - 2; } if(keyDown("left")) { ball.x = ball.x + 2; }

C: if(keyDown("right")) { ball.x = ball.x + 2; } if(keyDown("left")) { ball.x = ball.x - 2; }

D: None of the above

Answer:Option 3 (text)

Explaination:We want to move the ball to the right when right key is kept pressed so we have to add constant value in x direction and when we want the ball to the left when left key is kept pressed we have to subtract constant value in x direction. And correct syntax for key pressed event in code.org is keyDown("kay name");

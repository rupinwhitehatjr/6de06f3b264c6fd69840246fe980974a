While creating the world's hardest game we have to add a condition such that when the ding touches any dong the game the postion of ding should reset. Which of the following is a correct option?image

A: if(ding.Touching(dong1) || ding.Touching(dong2) || ding.Touching(dong3) || ding.Touching(dong4)) { ding.x = 40; ding.y = 190; }

B: if(ding.isTouching(dong1) || ding.isTouching(dong2) || ding.isTouching(dong3) || ding.isTouching(dong4)) { ding.x = 40; ding.y = 190; }

C: if(ding.bounceOff(dong1) || ding.bounceOff(dong2) || ding.bounceOff(dong3) || ding.bounceOff(dong4)) { ding.x = 40; ding.y = 190; }

The correct syntax for it is: if ((condition1) || (condition2) || (condition3) || (condition4)) { code; }

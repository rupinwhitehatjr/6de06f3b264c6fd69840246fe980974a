Q. What will be printed on the console when the following code snippet is run once?

A: 25 gets printed once

B: 25 gets printed twice

C: Nothing gets printed on the console.

D: The code is faulty and needs to be fixed.

Answer:Option 1 (text)

Explaination:"&&" condition return true only when both the conditions are true and "||" condition is executed when any one of the conditions is true. So the second condition is true and 25 is printed only once on the console.

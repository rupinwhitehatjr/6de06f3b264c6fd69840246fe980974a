Q. We want to add score to our game such that when the ball is missed by player paddle the computer score should increase by 1, choose the correct code block for this purpose.

We want to add score to our game such that when the ball is missed by player paddle the computer score should increase by 1, choose the correct code block for this purpose.image

A: if (ball.x < 0){ computerScore = computerScore + 1; }

B: if (ball.x > 0){ computerScore = computerScore + 1; }

C: if (ball.x < 400){ computerScore = computerScore + 1; }

D: if (ball.x > 400){ computerScore = computerScore + 1; }

Answer:Option 4 (text)

Explaination:Since the player paddle is on the right edge, if the ball's x position becomes greater than 400, it means the player missed hitting it. Automatically, it means it is a win for the computer player
